# Deliverance Toolkit

A Christ-Centered Resource for Spiritual Freedom - A web application for generating personalized deliverance prayers.

## Features

- **Standard Prayers**: Select from pre-written prayer templates
- **Custom Prayer Builder**: Build personalized prayers with specific details
- **Guided Session**: Step-by-step prayer generation with guided questions

## Deployment

This application is configured for deployment on Vercel. For detailed deployment instructions, see [DEPLOYMENT.md](DEPLOYMENT.md).

## Local Development

This is a static HTML/CSS/JavaScript application. To run locally:

1. Clone the repository
2. Open `index.html` in a web browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx serve
   ```

## API Configuration

The application supports integration with the following APIs:
